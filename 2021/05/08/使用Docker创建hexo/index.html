使用Docker创建Hexo

发表于 2021-05-08 更新于 2023-08-24

简介

​ 学习了在正常centos7环境中使用nginx和bt搭建hexo，也初步学习了docker，那么使用docker创建hexo的优势也显而易见了，比如使用docker的话备份，迁移和多地写post都成为了可能。那么就记录一下docker的折腾吧。

在阿里云上开发docker

使用阿里云的轻量应用服务器服务。选择中国（香港），兼容ping值和镜像下载速度，省去了配置镜像源的时间。选择Docker下最便宜的24元/月，用完即删。

在终端中使用镜像hexo：latest以交互模式启动一个容器。将主机的hexo_docker2文件夹 映射到容器内的hexo。 将主机的$HOME/.ssh 映射到容器内的 /root/.ssh。将容器的 4000 端口映射到主机的 4000 端口。并在容器内执行 /bin/bash 命令。docker会进入容器id为e8c22704c0b0的内部，

在这个环境下执行 hexo s 然后用浏览器打开：http://<服务器的公网IP>:4000/即可预览你的博客了。
